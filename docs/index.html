為什麼選擇 PyGMT？

純 Python 介面

使用熟悉的 Python 語法，無需學習複雜的 GMT 指令。

基於 GMT 核心

繼承 GMT 30年專業地圖繪製技術，品質有保障。

科學視覺化

完美支援地震、氣象、海洋、地質等領域圖表。

3D 立體圖

輕鬆繪製地形、地震斷層、海洋深度等3D視覺化。

完全開源

免費使用，社群活躍，持續更新中。

高效能

支援大數據集，快速渲染高解析度圖像。

只需 10 行程式碼，就能畫出專業地圖！ 
import pygmt

fig = pygmt.Figure()
fig.coast(
    region=[120, 123, 22, 26],
    projection="M15c",
    land="#d3d3d3",
    water="#87CEEB",
    shorelines="1/0.5p,black",
    frame="a"
)
fig.text(text="臺灣地圖 by PyGMT", x=121.5, y=24.5, font="18p,Helvetica-Bold,red")
fig.savefig("taiwan_map.png", dpi=300)
fig.show()

PyGMT vs 傳統 GMT

傳統 GMT

  • 複雜的 Shell 指令
  • 難以除錯與重複使用
  • 學習曲線陡峭
  • 不易整合其他 Python 套件

PyGMT 全新體驗

  • 純 Python 語法
  • 物件導向，易於除錯
  • 與 pandas、numpy 無縫整合
  • Jupyter Notebook 完美支援
