下載完整 Jupyter Notebook 範例

只需 10 行程式碼，就能畫出專業地圖！ 
import pygmt

fig = pygmt.Figure()
fig.coast(
    region=[120, 123, 22, 26],
    projection="M15c",
    land="#d3d3d3",
    water="#87CEEB",
    shorelines="1/0.5p,black",
    frame="a"
)
fig.text(text="臺灣地圖 by PyGMT", x=121.5, y=24.5, font="18p,Helvetica-Bold,red")
fig.savefig("taiwan_map.png", dpi=300)
fig.show()
